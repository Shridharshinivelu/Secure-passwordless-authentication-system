# Secure Passwordless Authentication System

## Objective
To develop a secure passwordless authentication system using Magic Link authentication.

## Features
- User Registration
- Email-based Magic Link Login
- Secure Token Verification
- Session Management
- User Dashboard

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- PHPMailer
- XAMPP

## Authentication Method
Magic Link Authentication

## Database
MySQL

## SMTP
Gmail SMTP / Mailtrap SMTP
